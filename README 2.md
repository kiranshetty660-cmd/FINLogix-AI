## 🎯 The Problem

Indian SMEs lose **₹4,000+ Crore annually** due to supply chain payment inefficiencies:

|Pain Point|Business Impact|
|-|-|
|🚨 Unpredictable supplier delays|Production halts, penalty clauses triggered|
|💸 Missed early payment discounts|2–3% savings left on the table every invoice|
|📉 Poor cash flow visibility|Working capital deployed inefficiently|
|⚠️ No early supplier risk warnings|Disruptions discovered after damage is done|

Finance and supply chain teams operate in silos — by the time a payment issue is visible, the damage is already done.

\---

## 💡 The Solution

**FINLogix AI** connects supplier data, payment schedules, and shipment records to **Google Gemini 1.5 Pro**. The result: a unified intelligence layer that predicts problems before they happen and tells you exactly what to do about them — in plain Hindi or English.

```
Raw Supply Chain Data  →  Gemini 1.5 Pro + ML Engine
         ↓
  Delay Predictions  +  Risk Scoring  +  Cash Flow Forecast
         ↓
  Payment Schedule Optimizer  →  Early Discount Capture
         ↓
  Natural Language Chat  →  One-Click Actions  →  Measurable Savings
```

## ✨ Features <a name="features"></a>

### 📊 Executive Dashboard

Real-time KPIs with AI-generated summaries — cash flow, on-time delivery rates, active risk alerts, and early payment savings. A **What-If Simulator** lets finance teams model payment scenarios with live Gemini-calculated projections.

### ⚙️ Optimization Engine *(New)*

Gemini continuously generates a **priority-ranked action plan** — each card shows the problem, the recommended fix, the financial impact, and a one-click execution button. No more spreadsheet hunting.

### 💸 Dynamic Early Payment Discounting

Three-tier discount engine that matches your real-time cash position to supplier discount windows, calculates annualized ROI for each opportunity, and schedules payments automatically.

### 🎮 Cash Flow What-If Simulator

Drag sliders to model different early payment percentages, discount rates, and payment windows. Gemini recalculates projected monthly and annual savings in real time.

### 🤖 Gemini Chat Interface

Ask questions in natural language — *"Which orders will be delayed this week?"* or *"Kaunsa supplier sabse zyada risky hai?"* — and get structured, data-backed answers with confidence scores and recommended actions.

### 🔍 Explainable AI (XAI)

Every prediction includes a **SHAP-style factor breakdown**: *"89% delay confidence — 40% supplier history + 35% route congestion + 15% monsoon signal + 10% port index."* No black boxes.

### 🏭 Supplier Risk Intelligence

All 47 suppliers scored on a 0–100 risk scale using payment history, delay patterns, financial signals, and external data (weather, port congestion). Visual score circles and AI flags per supplier.

### 📈 6-Month Cash Flow Forecast

AI-predicted inflow vs. outflow charts with annotated Gemini insights — surplus windows, deficit warnings, and optimal capital deployment moments.

### 📄 Gemini Report Builder

Generate custom PDF and Excel reports with natural language prompts: *"Monthly savings analysis with supplier breakdown for my CFO."*

## 🚀 Quick Start <a name="quick-start"></a>

### Option 1: Open the Prototype (No Setup Required)

```bash
# Clone the repository
git clone https://github.com/your-username/finlogix-ai.git
cd finlogix-ai

# Open in browser — works completely offline
open FINLogix\_AI\_Prototype.html
```

### Option 2: Full Stack (Coming Soon)

```bash
# Install dependencies
npm install

# Add your Gemini API key
cp .env.example .env
# Edit .env: GEMINI\_API\_KEY=your\_key\_here

# Run development server
npm run dev
```

### Try the Demo Data

1. Click **Upload Data** in the sidebar
2. Click **⬇️ Download Sample CSV** — a real file downloads
3. Re-upload it and watch Gemini process all 7 suppliers automatically
4. Explore the Dashboard, Optimization Engine, and Gemini Chat
